#BagAPI by NestJS

## Summary

It's an API for save data. 

- [x] Provide module for multiple database provider.
- [x] Provide module for upload files.
- [x] support middleware.
- [x] POO API.
- [x] Provider controller.
- [x] test unit.
- [x] add support swagger OpenAPI spec.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
